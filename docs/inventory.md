# 장표 인벤토리

모든 장표는 여기 등록한다. 등록 없는 장표는 존재하지 않는 것으로 취급하며, **복무(COURSE.md §1 Q1~Q6)** 가 비어 있는 장표는 deck-reviewer에서 FAIL이다.

## 상태 정의

| 상태 | 뜻 |
|---|---|
| 계획 | 구성만 확정, 미제작 |
| 초안 | HTML 작성됨, 검증 전 |
| 검증 | 브라우저 검증 통과(콘솔 0, 겹침 없음, 토큰 재생 확인) |
| 통과 | slide-reviewer PASS (완료 상태) |
| 대본 | 통과 + aside.notes 대본 작성됨 |
